## 1. 问题背景

自动驾驶车辆在复杂交通环境中，需要**预测其他车辆的交互行为**并做出决策。传统方法的问题是：
- 仅基于历史轨迹预测，**没有显式建模未来的交互**
- 自车（AV）被动响应预测结果，无法主动协调（如并道、左转等场景）

---

## 2. 核心思想：分层博弈论（Level-k Game Theory）

受认知科学启发，将交通参与者分为不同**推理层级**：

| 层级 | 行为特征 | 示例 |
|------|----------|------|
| **Level-0** | 完全独立，不考虑其他agent | "我只管自己开" |
| **Level-1** | 假设其他都是Level-0，并响应他们 | "我知道他会直走，所以我变道" |
| **Level-2** | 假设其他是Level-1，并响应他们 | "他知道我要变道，所以他会减速，那我就变道" |
| **...** | 迭代下去 | 更高阶的相互推理 | 

**关键洞察**：现实中人类驾驶通常只考虑1-2层推理，太高阶反而不稳定。

## 3. 模型架构

```
┌─────────────────────────────────────────┐
│  输入：场景信息（地图 + 所有车辆历史轨迹） │
└─────────────┬───────────────────────────┘
              ▼
┌─────────────────────────────────────────┐
│  Transformer 编码器                      │
│  - LSTM编码车辆历史状态                   │
│  - MLP编码地图信息                       │
│  - 自注意力建模场景元素关系               │
└─────────────┬───────────────────────────┘
              ▼
┌─────────────────────────────────────────┐
│  分层Transformer解码器                   │
│                                         │
│  Level-0解码器: 独立预测多模态轨迹        │
│       ↓                                 │
│  Level-1解码器: 看Level-0结果，交互推理   │
│       ↓                                 │
│  Level-2解码器: 看Level-1结果，进一步refine│
│       ↓                                 │
│  ... (可堆叠K层)                        │
└─────────────────────────────────────────┘
```

---

## 4. 关键技术细节

### (1) Level-0 解码
- 输入：初始模态查询（可学习的embedding）+ 场景编码
- 输出：每个agent的**多模态未来轨迹**（高斯混合模型 GMM）
  - 每个模态包含：均值位置(μx, μy) + 方差(σx, σy) + 概率分数

### (2) Level-k 交互解码 (k≥1)

核心操作：

1. **编码上一层轨迹**：用MLP将Level-(k-1)的轨迹编码为特征
2. **模态聚合**：用预测的分数做加权平均，得到确定性的未来特征
3. **自注意力交互**：让所有agent的未来特征互相"交流"（建模交互）
4. **掩码策略**：防止agent看到**自己**在上一层的未来（避免信息泄露）
5. **场景更新**：将交互特征拼回场景编码，做交叉注意力
6. **输出**：refined的GMM轨迹

### (3) 训练损失函数

#### ① 模仿损失（Imitation Loss）
- 用真实轨迹监督预测结果
- 负对数似然损失（NLL），只监督最接近真值的模态

#### ② 交互损失（Interaction Loss） - 关键创新
- 鼓励agent避开其他Level-(k-1) agent的轨迹
- 使用**排斥势场**（repulsive potential field）：
  - 当两车距离太近时，产生大损失
  - 实现安全避撞

## 6. 一句话总结

> **GameFormer用"分层博弈论"指导Transformer的层级解码结构，让模型像人类一样分层推理交互（我预判你的预判），同时用排斥势场保证安全性。**

## 8. 相关概念

- **Level-k Game Theory**: 分层博弈论，认知科学中的交互建模框架
- **GMM (Gaussian Mixture Model)**: 高斯混合模型，用于表示多模态轨迹预测
- **Transformer Encoder-Decoder**: 注意力机制-based的序列建模架构
- **Self-Attention**: 自注意力，用于建模agent之间的交互关系
