## 1. 问题背景

### 1.1 传统预测方法的局限

传统轨迹预测存在两个核心问题：

1. **仅基于历史信息**：传统方法只使用过去的轨迹来预测未来，忽略了自车（ego vehicle）未来行为对周围车辆的影响
2. **与规划解耦**：预测和规划是独立的模块，预测结果对不同的规划候选都是固定的

### 1.2 人类驾驶的启发

人类司机在驾驶时会进行 **"what-if" 思考**：
- "如果我激进地并道，其他车会怎么反应？"
- "如果我温和地并道，其他车又会怎么反应？"
- 基于不同假设的未来情景，选择最佳驾驶策略

**核心洞察**：人类将**自己的未来意图**作为条件，来预测周围车辆的反应。

## 2. 核心思想：规划感知预测 (Planning-Informed Prediction)

### 2.1 PiP  vs  传统方法

| 对比项 | 传统方法 | PiP (本文) |
|--------|----------|-----------|
| **输入** | 仅历史轨迹 | 历史轨迹 + 自车未来规划 |
| **输出** | 单一预测结果 | 对应每个规划候选的预测结果 |
| **交互建模** | 单向（他车→自车） | 双向（他车↔自车） |
| **适用场景** | 简单场景 | 高密度、高交互场景（如并道） |

## 3. 方法详解

### 3.1 网络架构

PiP 包含三个核心模块：

```
输入:
- 周围车辆历史轨迹 (H)
- 自车候选规划轨迹 (P)

          │
          ▼
┌───────────────────────┐
│  Planning-Coupled     │  ← 核心创新点
│  Module (PCM)         │
│  规划耦合模块          │
│                       │
│  • 编码历史交互       │
│  • 注入未来规划信息   │
│  • 生成社交张量       │
└───────────┬───────────┘
            │
            ▼
┌───────────────────────┐
│  Target Fusion        │
│  Module (TFM)         │
│  目标融合模块          │
│                       │
│  • 全卷积结构         │
│  • 多空间分辨率       │
│  • 建模目标间依赖     │
└───────────┬───────────┘
            │
            ▼
┌───────────────────────┐
│  Maneuver-based       │
│  Decoder              │
│  基于动作的解码器      │
│                       │
│  • 多模态分布输出     │
│  • 动作概率预测       │
└───────────┬───────────┘
            │
            ▼
输出: 每个周围车辆的
     未来轨迹分布
```

### 3.2 Planning-Coupled Module (PCM) - 规划耦合模块

**核心功能**：将自车规划信息注入预测过程

**具体实现**：
1. **历史编码**：用 LSTM 编码周围车辆的历史轨迹
2. **规划注入**：创建一个特殊通道，将自车的未来规划轨迹作为额外输入
3. **社交张量**：构建时空社交张量，融合历史和规划信息

**关键洞察**：
- 不只是把规划作为额外特征，而是构建**双时间域**（历史+未来）的交互建模
- 自车的不同规划会导致周围车辆不同的反应模式

### 3.3 Target Fusion Module (TFM) - 目标融合模块

**核心功能**：建模多个目标车辆之间的相互依赖

**设计动机**：
- 所有目标车辆的未来状态都通过自车规划紧密耦合
- 需要显式建模目标间的交互

**实现方式**：
- **全卷积结构 (Fully Convolutional)**
- **多空间分辨率**：在不同尺度上捕捉交互
- 类似于 U-Net 的编码器-解码器结构

### 3.4 Maneuver-based Decoder - 基于动作的解码器

**输出形式**：
- 对每个目标车辆，预测其未来轨迹的概率分布
- **多模态**：考虑多种可能的驾驶动作（如保持车道、变道、加减速等）
- 每个动作对应一条轨迹模式

**数学表示**：
```
P(Y | H, P) = Σ_k P(m_k | H, P) * P(Y | m_k, H, P)
```
其中：
- `Y`: 未来轨迹
- `H`: 历史轨迹
- `P`: 自车规划
- `m_k`: 第k个驾驶动作

## 4. 实验评估

### 4.1 数据集

| 数据集 | 场景 | 特点 |
|--------|------|------|
| **NGSIM** | 美国高速公路 | 45分钟，10Hz，拥堵/中等/畅通三种条件 |
| **HighD** | 德国高速公路 | 无人机航拍，110k+车辆，400-420米路段 |

**数据设置**：
- 观察窗口：3秒
- 预测窗口：5秒
- 采样频率：5Hz (输入) / 1Hz (规划输入)

### 4.2 对比基线

| 方法 | 类型 | 特点 |
|------|------|------|
| **S-LSTM** | 确定性 | Social LSTM，社交池化 |
| **CS-LSTM** | 确定性 | 卷积社交LSTM，基于动作的多模态 |
| **S-GAN** | 随机性 | GAN生成多样化轨迹 |
| **MATF** | 随机性 | 场景+智能体交互，GAN |
| **PiP-noPlan** | 消融 | PiP去掉规划耦合模块 |
| **PiP-noFusion** | 消融 | PiP去掉目标融合模块 |

### 4.3 定量结果

**评估指标**：
- **RMSE**: 均方根误差
- **NLL**: 负对数似然（衡量概率预测的准确性）

**主要结果** (NGSIM 数据集)：

| 方法 | RMSE (min) | NLL |
|------|------------|-----|
| S-LSTM | 2.41 | 2.45 |
| CS-LSTM | 2.15 | 1.98 |
| S-GAN | 1.93 | - |
| MATF | 2.05 | - |
| **PiP (本文)** | **1.53** | **1.05** |

**关键发现**：
1. PiP **显著超越**所有基线方法
2. 确定性模型 PiP 比采样3次的随机模型 RMSE 更低
3. 消融实验验证了 PCM 和 TFM 的有效性

### 4.4 定性分析：主动规划示例

**场景1：拥堵并道**
- 图4(a)：温和并道 → 安全完成
- 图4(b)：激进并道 → 与前车距离过近（危险）
- 图4(c)：更激进并道 → 预测到碰撞！

**场景2：高速换道**
- 图5(a)：快速右转跟随前车 → 预测到后车追尾碰撞
- 图5(b)：右转超车 → 预测到碰撞
- 图5(c)：适当减速后右转 → 安全完成
- 图5(d)：左转 → 相对容易（左侧空间更大）

**重要发现**：
- PiP 能预测**训练数据中未出现的碰撞情况**
- 证明了模型的泛化能力

## 5. 与 GameFormer 的对比

| 维度 | PiP (Song 2020) | GameFormer (Huang 2023) |
|------|-----------------|------------------------|
| **核心思想** | 条件预测（自车规划→他车反应） | 分层博弈（Level-k推理） |
| **交互建模** | 双向但单向传播（ego→others） | 迭代双向（多轮Level-k） |
| **输入条件** | 自车规划候选轨迹 | 场景历史状态 |
| **推理层级** | 单层（ego的规划影响others） | 多层（Level-0,1,2...） |
| **输出形式** | 条件化的多模态轨迹 | 层次化的多模态轨迹 |
| **架构** | CNN+LSTM+Planning耦合 | Transformer Encoder-Decoder |
| **适用场景** | 高速公路并道/换道 | 城市路口交互 |

**关系**：
- **PiP 是 GameFormer 的前置工作之一**
- GameFormer 引用了 PiP 作为 "conditional prediction" 的代表性方法
- 两者都解决**预测-规划耦合**问题，但方法不同：
  - PiP：直接条件化预测过程
  - GameFormer：通过分层博弈迭代推理

---

## 6. 一句话总结

> **PiP 通过将自车规划信息注入预测过程，实现了"what-if"式的条件预测，使得预测结果能对应不同的规划候选，从而为高密度交互场景下的安全规划提供了更准确的评估依据。**

---

## 7. 核心贡献回顾

1. **Planning-Coupled Module**: 首次将自车规划显式引入预测过程
2. **Target Fusion Module**: 全卷积结构建模多目标间的耦合关系
3. **Planning-Prediction-Coupled Pipeline**: 新的预测-规划耦合流程
4. **SOTA 性能**: 在 NGSIM 和 HighD 数据集上超越现有方法
